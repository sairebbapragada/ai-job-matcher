# 🚀 AI Job Matcher

An AI-powered job matching platform that helps students find the best job opportunities based on their resume and skills.

---

## 🔥 Features

- 📄 Resume input and skill extraction  
- 🤖 AI-based job matching  
- 📊 Match score visualization  
- 📈 Insights into best-fit roles  

---

## 🧠 Tech Stack

- Python  
- SQL (SQLite)  
- Streamlit  

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app/app.py
